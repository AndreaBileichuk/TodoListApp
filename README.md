This is a comprehensive Task Management application designed to help users organize their daily activities. The solution consists of two main components: an ASP.NET Core MVC web application for the user interface and a Controller-based ASP.NET Core Web API for backend data processing.

The system utilizes PostgreSQL for data storage, Entity Framework Core for ORM, and ASP.NET Core Identity for secure user authentication and authorization.
